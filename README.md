# "Hola Mundo" en Java

## Descripción

Programa "Hola Mundo En Java"

## Compilación y ejecución del programa

Para **compilar** el programa teclea lo siguiente:

```console
javac holaMundo.java
```
Y para **ejecutarlo**:

```console
javac holaMundo
```
